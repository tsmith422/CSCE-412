\doxysubsection{Detailed Description}
Simulates a load balancer that distributes requests to multiple web servers. 

The \mbox{\hyperlink{classLoadBalancer}{Load\+Balancer}} manages a queue of incoming requests and a pool of web servers. It handles request assignment, ticking server time, and simulating system behavior over multiple clock cycles with logging support.
