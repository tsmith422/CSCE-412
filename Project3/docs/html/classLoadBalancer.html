Detailed Description

Simulates a load balancer that distributes incoming web requests to multiple web servers.

The LoadBalancer manages a queue of incoming requests, a dynamic pool of web servers, and tracks statistics such as rejected requests and processed requests. It supports scaling the number of servers based on the load.

void LoadBalancer::scaleServers ( )

Dynamically adjusts the number of web servers based on load conditions. Adds or removes servers to maintain balanced capacity.

Dynamically adds or removes servers based on current queue size to maintain balance. Adds a server if queue size is above 500 and under max server limit (20). Removes an idle server if queue size below 100 and above min server limit (5).
